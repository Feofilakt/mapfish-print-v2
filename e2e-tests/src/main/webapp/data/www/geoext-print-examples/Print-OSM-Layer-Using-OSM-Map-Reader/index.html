Printing a OSM Layer

This example prints New York using an OSM Layer.

For context, the roads of New York overlay the OSM layer in blue. This is intended to \ verify that the two layers are correctly lining up.

Click here to see the config.yaml

The js is not minified so it is readable. See print-page.js.
