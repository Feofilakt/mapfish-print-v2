Printing a WMTS Layer

This example prints the New York Tiger layer group using WMTS Version 1.0.0.

For context, the roads of New York overlay the WMTS layer in blue. This is intended to \ verify that the two layers are correctly lining up.

Click here to see the config.yaml

The js is not minified so it is readable. See print-page.js.
